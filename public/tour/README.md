# 360° Panorama Images

## Quick Setup

Replace the placeholder images with actual 360° panoramas:

1. **Download free samples:**
   - Flickr Equirectangular Group: https://www.flickr.com/groups/equirectangular/
   - Poly Haven HDRIs: https://polyhaven.com/hdris

2. **Required images:**
   - `entrance.jpg` (2:1 ratio, e.g., 4096×2048px)
   - `living.jpg` (2:1 ratio, e.g., 4096×2048px)
   - `bathroom.jpg` (2:1 ratio, e.g., 4096×2048px)

3. **Image format:**
   - Type: Equirectangular projection
   - Ratio: 2:1 (width:height)
   - Recommended size: 4096×2048px or 8192×4096px
   - Format: JPEG (for smaller file size)

## Taking Your Own 360° Photos

### Option 1: Professional Camera
- Ricoh Theta Z1 (~$1000)
- Insta360 One X2 (~$400)

### Option 2: Smartphone
- Google Street View app (iOS/Android) - Free!
- Panorama mode + manual stitching

### Tips:
- Use a tripod at ~1.5m height
- Shoot in bright, even lighting
- Keep the camera level
- Avoid moving objects in frame
